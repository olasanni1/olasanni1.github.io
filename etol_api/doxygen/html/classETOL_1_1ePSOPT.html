◆ setup()

void ETOL::ePSOPT::setup ( )
virtual

Constructs an optimization problem from settings.

Sets all initial guesses to 0, except the time guess. The time guess is a equally set from 0 to dt*nstep. Configures the algorithm to use IPOPT to solve a NLP problem that is formed from enforcing constraints at Legendre nodes and with derivative computed with automatic differentiation.

Implements ETOL::TrajectoryOptimizer.

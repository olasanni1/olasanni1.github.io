Detailed Description

Loads a ETOL Config XML file and uses PSOPT to solve a ETOL.

Version
1.0.1

DESCRIPTION

Solves the following OCP minimize integral (u0(t) *u0(t) + u1(t) *u1(t)) from t0 to tf subject to dx/dt = u0(t) for t in {t0,tf} dy/dt = u1(t) for t in {t0,tf} <x(t) y(t)> notin Obstacle for t in {t0,tf} <x(t),y(t)> notin Moving Obstacle for t in {t0,tf}

Detailed Description

Loads a ETOL Config XML file and uses GLPK to solve a ETOL.

DESCRIPTION

Solves the following MILP minimize sum(u_k_2 + u_k_3) for k = 0 to NSteps subject to x_k = x_(k-1) + dt * u_k_0 for k = 0 to Nsteps y_k = y_(k-1) + dt * u_k_1 for k = 0 to Nsteps u_k_2 = |u_k_0| for k = 0 to Nsteps u_k_3 = |u_k_1| for k = 0 to Nsteps (x_k, y_k) notin Obstacle for k = 0 to Nsteps (x_k, y_k) notin Moving Obstacle for k = 0 to Nsteps
